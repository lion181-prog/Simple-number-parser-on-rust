This is a simple project for printing numbers in Rust.

Here's an example of it running:

@user ➜ /workspaces/Simple-number-parser(main) $ cargo run
Compiling Simple-number-parser-on-rust v0.1.0 (/workspaces/Simple-number-parser-on-rust)
Finished `dev` profile [unoptimized + debuginfo] target(s) in 1.84s
Running `target/debug/Simple-number-parser-on-rust`
10 10 10
i=10
j=10
k=10

where 10 10 10 is where you enter numbers, and the output is the saved values ​​in variables 
(when you stop the project, these values ​​will be reset)
